## Dataset Introduction
This is a measured dataset for reinforced concrete rebar detection based on Electromagnetic Tomography (EMT), which contains a total of 9100 valid experimental samples.

### Experimental Conditions
1. Rebar layout: Single rebar and double rebars
2. Rebar diameter: Five specifications including 12 mm, 14 mm, 16 mm, 18 mm and 20 mm
3. Spacing settings for double rebars
   - Horizontal spacing: 60 mm, 90 mm, 120 mm
   - Vertical spacing: 5 mm, 10 mm, 15 mm

### Data Classification
Considering the common problem of data loss in on-site detection, all samples are divided into five categories according to the data missing rate.
